# Quantum Random Number Generator using Qubits

This project generates random bits (0 or 1) using quantum computing (qubits) and shows the result in a bar graph.

## Files
- abstract.pdf → Project Abstract
- presentation.pptx → Project PPT
- code.py → Project Code

## How to Run
1. Install Python
2. Install required libraries (qiskit, matplotlib)
3. Run code.py
